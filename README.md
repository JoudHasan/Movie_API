5. ### ğŸ”— API Endpoints

Here are the main API endpoints available in this project:

#### ğŸ¬ Movies

- **GET** `/movies`: Get a list of all movies.
- **GET** `/movies/title/:Title`: Get data about a single movie by its title.
- **GET** `/movies/genre/:Genre`: Get data about movies by genre.

#### ğŸ‘¤ Users

- **GET** `/users`: Get a list of all users.
- **GET** `/users/:Username`: Get data about a user by username.
- **POST** `/users`: Register a new user.
- **PUT** `/users/:Username`: Update a user's information.
- **POST** `/users/:username/favorites/:movieId`: Add a movie to a user's favorite list.
- **DELETE** `/users/:Username/movies/:MovieID`: Remove a movie from a user's favorite list.
- **DELETE** `/users/:Username`: Delete a user.
