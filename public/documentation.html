My Flix app

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Get a list of ALL movies /movies GET none A JSON object holding an array with all movies.
Get data about a single movie by title /movies/:Title GET None A JSON object holding data about a single movie, containing description, genre, director, image URL, and whether its featured or not property. Example: 
                            
                            {
                                "Title": "The Lives of Others",
                                "Description": "In 1984, when an East German Stasi agent is ordered to spy on a writer...",
                                "Genre": {
                                    "Name": "Drama",
                                    "Description": "In film and television, drama is a category or genre of narrative fiction..."
                                },
                                "Director": {
                                    "Name": "Florian Henckel"
                                },
                                "ImageURL": "https://images.app.goo.gl/tukE2gj2t1MPqeL7A",
                                "Featured": true
                            }
Get data about a genre by genre's name /movies/genre/:genreName GET none A JSON object holding data about a genre containing name and description properties. Example:

                                }

Add a new user /users POST A JSON object holding user data, structured like this: 
                            
                                {
                                    "id": "650a80a5940ff76d61be79dc",
                                    "userName": "john",
                                    "Email":"john@gmail.com",
                                    "password": "john123",
                                    "favoriteMovies": ["the lives of others"]
                                }
A JSON object holding user data plus an ID and an array for placing favorite movies: 
                            
                        {
                            "id": "650a80a5940ff76d61be79dc",
                            "userName": "john",
                            "Email":"john@gmail.com",
                            "password": "john123",
                            "favoriteMovies": ["the lives of others"]
                        }
Update user info /users/:id PUT A JSON object holding the user info that should be changed. It should contain at least one of the following key/value pairs: 
                            
                        {
                            "username": "New Username",
                            "Email":"New Email",
                            "password": "New Password"
                            "favoriteMovies": ["the lives of others"]
                        }
A JSON object holding all user info with all key/values pair updated: 
                            
                        {
                            "id": "650a80a5940ff76d61be79dc",
                            "username": "New Username",
                            "Email":"New Email",
                            "password": "New Password",
                            "favoriteMovies": ["The Lives of Others"]
                        }
Add a movie to the user's favorite list /users/:id/:movieTitle POST None A JSON object holding all user info with all key/values pair updated: 
                            
                        {
                            "id": "650a80a5940ff76d61be79dc",
                            "username": "john",
                            "Email":"john@gmail.com",
                            "password": "johnd123",
                            "favoriteMovies": ["The Lives of Others"]
                        }
Remove a movie from the user's favorite list /users/:id/:movieTitle DELETE None A JSON object holding all user info with all key/values pair updated: 
                            
                        {
                            "id": "650a80a5940ff76d61be79dc",
                            "username": "john",
                            "Email":"john@gmail.com",
                            "password": "john123",
                            "favoriteMovies": ["The Lives of Others"]

Delete a user /users/:id DELETE None A JSON object confirming the user deletion.
