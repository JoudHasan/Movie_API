My Flix app

Movie API App The Movie App is an exciting and user-friendly platform designed to cater to all your movie-related needs. Whether you're a cinephile searching for the latest blockbusters or someone looking for hidden cinematic gems, our app provides a comprehensive movie-watching experience.

Endpoints

Request URL Response
GET /movies JSON array of all movie data
GET /movies/:title JSON object containing data about a single movie by title
GET /movies/search/:genre JSON array of all movie data by genre
GET /movies/search/:director JSON array of all movie data by director
GET /movies/search/:year JSON array of all movie data by year
GET /movies/search/:actor JSON array of all movie data by actor
GET /users JSON array of all user data
GET /users/:username JSON object containing data about a single user by username
POST /users JSON object containing data about a single user by username
PUT /users/:username JSON object containing data about a single user by username
POST /users/:username/favorites/:movieID JSON object containing data about a single user by username
DELETE /users/:username/favorites/:movieID JSON object containing data about a single user by username
DELETE /users/:username JSON object containing data about a single user by username
